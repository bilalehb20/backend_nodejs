  QUICK TESTS (Test deze eerst!)
═══════════════════════════════════════════════════════════

1. API Root:
   GET http://localhost:3000/api
   → Verwacht: JSON met endpoints overzicht

2. Users:
   GET http://localhost:3000/api/users
   → Verwacht: [] of array met users

3. Events:
   GET http://localhost:3000/api/events
   → Verwacht: JSON met events array + pagination

═══════════════════════════════════════════════════════════
  BELANGRIJKSTE TIP
═══════════════════════════════════════════════════════════

Check altijd de prefix in src/app.js:
- app.use('/api/users', usersRoutes) → URL = /api/users
- app.use('/api/events', eventsRoutes) → URL = /api/events
- app.use('/api/auth', authRoutes) → URL = /api/auth
